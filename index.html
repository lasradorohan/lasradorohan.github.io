    GAN based Generative Face Completion

    This project involved finding a way to generate the missing parts in images from the existing sections. To simplify the task a bit, I decided to fix the region which was missing, specifically the right half of images.

    This was an individual project to explore the working of GANs.

    At this time, I had learnt about the ability of certain neural networks to generate more data similar to existing data, mostly GANs. More commonly GANs were used to unconditionally generate similar looking images. However, my aim was to generate missing halves such that it matches with the existing half.

    I also experimented with LSTMs for this task, as LSTM had been previously used for various tasks involving generating novel sequences, but didn't achieve good results.
    This is when I came across Pix2Pix GANs, a neural network architecture able to generatively map images to images.

    Pix2Pix GANs consist of 2 competing networks. A U-Net generator, which is like a convolutional auto-encoder but with skip connections between encoding and decoding layers of same shape, originally used for the biological segmentation task. And a PatchGAN discriminator, a convolutional downsampling network being trained to discriminate between the generator's output and the ground truth.

    With the help of some tutorials on tensorflow.org, I coded the data input pipeline and the model architecture for the Pix2Pix GAN. I trained it on the CELEB-Faces dataset, splitting the images in halves. It was able to generate visually satisfactory results.

    Deep learning-based approach for correction of wide-angle lens distortion

    Trained a Convolutional Neural Network to determine the wide-angle lens distortion coefficients in input images and used them to warp pixels back to their correct positions to undistort the images.

    This project involved the detection and correction of wide angle distortion in single images without any additional information about the capturing device. This distortion is commonly recognized as a "barrel" effect, especially in fisheye lens cameras that try to capture a larger Field of View while sacrificing distortion effects.

    The idea for this project came about from the knowledge of the ability of Convolutional Neural Networks to detect various artifacts/distortions in images, without focusing on the content of the image.

    The first task was to learn about the mathematical process modelling said distortion. One such well-known process was Fitzgibbon's division model [Fitzgibbon, A. W. (2001)]

    Further modifications were made to the formula to include a scale factor so that the distorted image does not contain any black boundaries The second task was to find a dataset that would provide images distorted with different intensities and their corresponding coefficients. As to the best of my googling, I couldn't find one, I decided to create my own.

    I used the Indoor Scene Recognition dataset ( http://web.mit.edu/torralba/www/indoor.html) and manually randomly distorted the images to generate one. However in the process I had to discard around 7k images that were too blurry.

    The next step was to train a CNN model to predict the coefficient.
    For this I resized the images to 512×512, used XCeption network as the backbone, multiplied feature maps with a radial matrix, and added further convolution and dense layers to get the coefficient.

    A simple interface was made with StreamLit python library, adding to the usability of the solution.

    Image Manipulation Utility written in CUDA

    Implemented a GPU accelerated command-based image manipulation tool using the CUDA computing platform API in C++. The tool allows queuing up multiple commands and execute them together on a batch of images.

    This project was performed for exploring the low level working of general purpose computation on the GPU using the CUDA GPGPU API in C++. For this, I selected the task as developing an image manipulation utility.

    A team of four with I as the lead wrote the code for allocating memory, copying data over to the GPU, the GPU kernels executing block-wise on different SMs on the GPU, copying back the results and deallocating. This was done for each image command: rotation, gamma correction, radial distortion, contrast, sharpening, blur, skew and conversion to black and white.

    I individually was also responsible for writing the command line based interface which allowed selecting multiple images, creating a pipeline consisting of multiple image commands, lazily loading the images, viewing and storing.

    Music recommendation based on facial expressions

    Created a web app that detects the user’s expression via the webcam and recommends songs to suit the user’s mood. Facial expression detection is implemented in TensorFlow.js for the app to run locally in the browser.

    This project was performed to explore deployment of deep learning solutions in the web scenario, such that a user may be able to run the application locally in their browser.
